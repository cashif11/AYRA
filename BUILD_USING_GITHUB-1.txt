STEP 5: Add GitHub Actions Build Workflow
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

This is what builds your APK automatically!

In GitHub:
1. Click "Add file"
2. Click "Create new file"
3. Name it: .github/workflows/build-apk.yml
4. Copy-paste this content:

---BEGIN CODE---

name: Build APK

on:
  push:
    branches: [ main, master ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - uses: actions/checkout@v3
    
    - name: Setup Java
      uses: actions/setup-java@v3
      with:
        distribution: 'zulu'
        java-version: '11'
    
    - name: Setup Flutter
      uses: subosito/flutter-action@v2
      with:
        flutter-version: '3.16.0'
        channel: 'stable'
    
    - name: Get dependencies
      run: cd ai_studio_pro && flutter pub get
    
    - name: Build APK
      run: cd ai_studio_pro && flutter build apk --release
    
    - name: Upload APK
      uses: actions/upload-artifact@v3
      with:
        name: AI-Studio-Pro-APK
        path: ai_studio_pro/build/app/outputs/flutter-apk/app-release.apk

---END CODE---

5. At bottom, click "Commit new file"
